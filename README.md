# ☕ Java Learning Repository

This repository contains my **Java learning practice programs**.  
I created this repository to store the Java concepts I learned while studying and practicing programming.

It includes simple programs that help beginners understand the **basic concepts of Java programming**.
---

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sanjayn29/Java.git
```

### 2️⃣ Navigate to the Folder

```bash
cd Java
```

### 3️⃣ Compile the Java File

```bash
javac Hello.java
```

### 4️⃣ Run the Java Program

```bash
java Hello
```
